# Agent instructions

tuicade is a small terminal arcade with simple games, 
built to demonstrate **parallel agent sessions**.

- Read the Makefile.
- Use `make` to install and lint and test.
- Use `make` to check if linting and tests pass. Fix issues.
- If you want to run the app: use `uv run tuicade`
- Press `q` inside any game to return to the launcher.
